# Personal Portfolio Website
 
A personal portfolio website built with HTML, CSS, and JavaScript, showcasing my projects, skills, and experience as a final-year Computer Science student at Kingston University.
 
## 🌐 Live Site
 
[savio-rodrigues.netlify.app](https://savio-rodrigues.netlify.app/)
 
## ✨ Features
 
- **Dark / Light mode** toggle for comfortable viewing
- **Project filter** to browse work by category
- **Animated skills section** with visual progress indicators
- **Contact form** powered by EmailJS — no backend required
- Fully responsive layout across desktop and mobile

## 🛠️ Tech Stack
 
- HTML5, CSS3, Vanilla JavaScript
- EmailJS (contact form)
- Hosted via Netlify
